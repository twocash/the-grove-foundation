# Mockup Screenshots

The PNG screenshots are too large to embed in the repo. 

## How to View
1. Open the HTML files in a browser:
   - `mockup_4_dark.html` — Dark mode chat styling
   - `mockup_3_light.html` — Light mode chat styling

2. The HTML files are fully functional mockups that render exactly like the screenshots would show.

## Key Patterns to Extract

### Dark Mode (mockup_4_dark.html)
- User messages: `bg-primary text-white rounded-2xl rounded-tr-sm`
- AI messages: `bg-surface-dark text-slate-100 border border-border-dark rounded-2xl rounded-tl-sm`
- Input: `bg-surface-dark border-border-dark focus-within:border-primary`
- Colors: primary=#197fe6, background-dark=#111921, surface-dark=#1E293B, border-dark=#293038

### Light Mode (mockup_3_light.html)
- User messages: `bg-primary text-white rounded-2xl rounded-tr-none`
- AI messages: `bg-[#f1f5f9] text-[#0e141b] rounded-2xl rounded-tl-none`
- Input: `bg-[#f8fafc] border-[#e2e8f0]`
- Max content width: `max-w-4xl mx-auto`
