## Comparison: Welcome vs. Switch

| Element | WelcomeInterstitial | LensPicker (Switch) |
|---------|---------------------|---------------------|
| Header | Terminal version + "Connection established" | Terminal version only |
| Main Title | "Welcome to The Grove." | "Switch Lens" |
| Body Copy | Full welcome paragraph | None (just subtitle) |
| Subtitle | "Select Your Starting Lens" | "Change your perspective..." |
| Lens Grid | Full grid + Create Your Own | Full grid + Create Your Own |
| Footer | "You can switch anytime..." | "Your current lens shapes..." |
| Visual Weight | Heavy (establishes context) | Light (utilitarian) |
